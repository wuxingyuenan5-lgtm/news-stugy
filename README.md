# Cross-Asset Information Briefing

面向个人阅读的跨资产信息整理与日报、周报生成项目。

覆盖范围：

- 宏观与流动性；
- 美股；
- A股与港股；
- 贵金属；
- 加密资产；
- 能源市场；
- AI与前沿科技。

## 第一版目标

第一版只解决三个问题：

1. 应该持续关注哪些信息源；
2. 每天如何筛选、分类和整理信息；
3. 日报、周报以什么内容和格式呈现。

最小工作流：

```text
信息源清单
→ 收集候选信息
→ 去除明显重复与低价值内容
→ 按市场和内容类型分类
→ 人工选择重点信息
→ 按固定模板生成日报、周报
→ 保存历史报告并持续优化
```

## 第一版交付物

- 信息源清单与分级规则；
- 信息条目的最小字段规范；
- 每日收集箱模板；
- 日报模板；
- 周报模板；
- 人工筛选与编辑规则；
- 两个完整交易周的试运行复盘。

## 设计原则

1. 先把信息源和报告成品做稳定，再考虑自动化和研究增强。
2. 所有重要信息必须能够回到原始来源。
3. AI优先承担摘要、翻译、分类和格式整理，不替代最终筛选。
4. 日报强调增量信息，周报强调归纳和持续跟踪。
5. 每增加一个字段或模块，都应证明它能改善阅读效率或报告质量。
