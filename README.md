# Cross-Asset Research OS

面向个人日报、周报与持续研究的跨资产信息整理系统。

覆盖范围：

- 宏观与流动性
- 美股
- A股与港股
- 贵金属
- 加密资产
- 能源市场
- AI与前沿科技

项目目标不是简单聚合新闻，而是完成：

> 信息采集 → 事实核验 → 事件结构化 → 预期差识别 → 跨资产传导 → 观点更新 → 日报/周报输出 → 历史复盘

## 当前阶段

项目处于架构设计阶段。第一版先建设一个可人工审核、可逐步自动化的 MVP，不直接追求全自动研究。

## 设计原则

1. 原始事实、分析判断和最终观点分层保存。
2. 每条重要信息必须保留来源、时间、预期差和影响路径。
3. 日报关注增量变化，周报关注状态切换与观点验证。
4. AI负责提取、分类、去重和辅助推理，人负责重要性判断与最终结论。
5. 先人工跑通工作流，再逐步自动化。
